Save and export a document
To save your Patternshop file, use the "File » Save" menu option, or the "⌘ + S" keyboard shortcut, then choose the file name. This will be saved in .pts format (Patternshop document), and you can then load it using the "File » Open" option or the keyboard shortcut "⌘ + O".

To export the network to PNG format, use the "File » Export the document" menu option, then choose a file name. It may take several seconds to export a file, depending on the size of your document; its weight and format depend on the dimensions configured when creating the document. Note: It is not possible to export a document with the evaluation version of the software.

To print your document, use the "File » Print" menu option, then configure the printing options. Note: The demo version of the software does not allow for printing.
