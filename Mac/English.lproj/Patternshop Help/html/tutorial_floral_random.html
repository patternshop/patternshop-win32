Create a random floral design

Using the same principle we will create a "random" network based on 3 patterns.

Open a new document (⌘ + N)

A dialog box will ask you to enter the width and height of your document and its resolution.

Enter "A4" into the predefined formats.

Import the Poppies 1 image available in the image bank (⌘ + D)

In the same pattern, then import the Poppies B and Poppies C image.

Create two new patterns: (⌘ + M) and drag them inside the Project window in order to have only one image per pattern:

Move and transform your patterns so as to optimise the random impression:

Uncheck the "Display the selection box" checkbox to see your network without the selection boxes of the patterns.
