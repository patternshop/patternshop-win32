Manipulate a pattern
Using the arrow tool, rotate the pattern 45° or enter 45° in the transformations window (see "The Transformation window").


Place the cursor outside the pattern, close to one of the handles located on the corners, until it takes the shape of a double curved arrow. You can then change the orientation of the pattern by clicking and moving the mouse.

Note: Changing the orientation of the pattern directly affects the layout of the image network:


Proceed in the same way, this time placing the cursor on the handles and not the side, this time to modify the dimensions of the pattern.

Note: Reducing the pattern will bring the the images closer together and make the network denser.


By passing the arrow over the edges of the pattern frame, a hand appears, which means that you can can warp the pattern by applying torsion, turning it into a parallelogram. You can also use, in the transformations window, the fields Torsion H (horizontal) and V (vertical) given in degrees.


Comment: Applying torsion to the pattern also affects the layout of the network:


Reset the shape of the pattern in the menu "Pattern » Reset pattern", then select the image and duplicate it, using the option "Edit » Duplicate" or the keyboard shortcut "⌘ + R". The new duplicated image appears in the Project window under the associated pattern.


Finally you can use the "Split" section in the transformations window (see "The Transformation window") to split the pattern into sections.


These will be useful if you want to rigorously place several images in a pattern. You can now, holding shift down, fix the image to the intersections of the grid.


You can then move on to stage 5 to change the background colour of your document.
