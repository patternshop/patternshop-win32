Create a Pattern
Create a pattern in your document, using the option on the "Pattern » Create a new pattern" menu, or the "⌘ + M" keyboard shortcut.


A box appears in the document: this is the pattern in which you can import your images, and which is used to control their layout. You can deactivate the fill of the box or change its colour on the "Pattern" menu, so as to make it more or less visible.

Then use the arrow tool (see "The Tools window") or modify the values in the transformations window (see "The Transformation window") to rescale it, reposition it, rotate it or apply torsion to the pattern.

Once the changes to the pattern have been completed, you can move on to stage 3.
