Import a template

Patternshop patterns give you a precise view of your work applied to different formats, and lets you best adjust it to the size of the network.

Create a network with the image(s) of your choice.

 

In the File > Import a template menu…  import your template in Patternshop.

Note, you may only have one template per document. If you have already imported one, It will be replaced.

The imported template will appear in the "Project" window.

By clicking on the rectangular area of the project window, you can open the Colour selector to define a background colour for your template.

You can then see the result.
