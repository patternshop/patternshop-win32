La fenetre "Outils"
Vous trouverez dans la fenêtre "Outils" les différents éléments disponibles pour modifier le document.


La flèche :
La flèche permet de séléctionner, de déplacer et de modifier la taille ou la forme d'une matrice, d'un motif ou d'une image. Cliquez sur un objet pour le séléctionner, et faire ainsi apparaitre le cadre de séléction :


Le fait de séléctionner un motif, comme dans l'illustration ci-dessus, sélectionne automatiquement la matrice qui le contient. Il est alors possible de déplacer un objet en cliquant dessus puis en déplaçant la souris tout en maintenant le bouton enfoncé. Les poignées servent, elles, à modifier la taille et la forme des objets; selon la position de la souris le pointeur changera d'aspect, indiquant les différentes actions possibles : pivoter, redimentionner et, pour les matrices uniquement, modifier l'inclinaison.

La main :
La main est utilisée pour se déplacer dans le document. Cliquez puis déplacez la souris tout en maintenant le bouton enfoncé pour déplacer la vue dans la direction souhaitée (ceci est utile quand le document n'est pas visible en entier dans la fenêtre).

La loupe :
Pour zoomer dans le document, vous pouvez utiliser la loupe. En cliquant puis en déplaçant la souris tout en maintenant le bouton enfoncé, le niveau de zoom sera respectivement augmenté ou réduit si le mouvement de la souris était orienté vers le haut ou vers le bas.
Astuce : Pour rétablir un niveau de zoom de 100% (taille réelle), effectuez un double clic sur l'outil "Loupe".
