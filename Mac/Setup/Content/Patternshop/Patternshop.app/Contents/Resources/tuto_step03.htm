Importing an image into a pattern
To import an image into a pattern, select it, then use the "File » Import an image" option, or the keyboard shortcut "⌘ + D". Then choose your image using the dialog box which appears.
Note: To make the most of the possibilities available in Patternshop, we recommend you import images using png file format to preserve transparency (8bit or 24bit png).

The image is imported into the pattern and is directly repeated in the network:


The image also appears in the Project window (see "The tracing pattern window") in the associated pattern folder:


Using the Transformations window (see "The Transformations window"), make this image horizontally symmetrical by preceding with a (-) symbol the value indicated in pixels, or in % in the Dimension W (width) field. To ensure vertical symmetry, you should precede with a (-) the value indicated in the Dimension H (height) field.

Comment: In a classic layout like this, Patternshop lets you move the images by using the keyboard arrows, thus allowing for extremely precise control of the distances between each motif, such as to find the ideal layout.
