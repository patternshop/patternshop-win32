Create a new document
To create a new document, use the "New" option on the "File" menu, or the keyboard shortcut "Ctrl + N". A dialog box then invites you to enter the width and height of your document, as well as its resolution.


Attention : This determines the document output size, onscreen and printed. You can also choose from among a list of predefined formats, A4, A3, B5, etc.

Once the settings have been entered, confirm with the "OK" button and a blank document will be created. You can then move on to stage 2.
