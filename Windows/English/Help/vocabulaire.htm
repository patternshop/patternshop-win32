Vocabulary
Here is a list of definitions of the key terms used in Patternshop.

  • Image (a) : an image that makes up the network.
  • Pattern (b) : Adjustable area in which the images are placed, and which defines their layout in the network.
  • Free image : unrepeated image which can be placed anywhere on the document.
  • Network (c) : Unlimited space, formed by the repetition of motif/s contained in patterns.
  • Document (d) : Defined area of a network.
  • Template : Predefined form on which the network is applied.
