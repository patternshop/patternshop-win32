The "Transformation" window
The "Transformation" window lets you arrange and rescale objects, as well as apply rotation and torsion.


Just modify the value contained in one of the editing boxes to make the change. The "Split" section lets you divide the pattern into sub divisions (values must be at least 2 to have any effect), and thus makes it possible to position images precisely (keep the "shift" key pressed while moving a image so that it aligns to the grid of the pattern that contains it).

The "Link images to the pattern" box has the effect of applying all modifications made to a pattern to the images it contains (rescaling, rotation). This option can also be activated when necessary by keeping the "Ctrl" key pressed during manipulation.
