The "Preferences" window

In preferences you can define the default size of the pattern, but also its minimum size to avoid slowing down the application with too many repeated images.

By default the minimum size of the pattern is set at 50 x 50 pixels.

You can also, for the same purpose of optimising performance, limit the maximum size of windows. This is highly advised for high screen resolutions.
