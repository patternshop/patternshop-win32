Importer un motif dans une matrice
Pour importer un motif dans une matrice, sélectionnez la, puis utilisez l'option du menu "Fichier » Importer un motif", ou le raccourci clavier "Ctrl + D". Choisissez ensuite votre motif dans la boite de dialogue qui apparait.
Attention : Pour tirer le meilleur parti des possibilités de Patternshop, nous vous recommandons d’importer des motifs détourés au format png pour préserver la transparence (png 8bits ou 24bits).

Le motif apparaît aussi dans la fenêtre des Calques (voir "La fenêtre des Calques") sous la matrice associée :


En utilisant la fenêtre Transformations (voir "La fenêtre Transformation"), effectuez une symétrie horizontale de ce motif en précédent d’un signe (-) la valeur indiquée en pixel ou en % dans le champ Dimension L (largeur). Pour effectuer une symétrie verticale, il faut précéder d’un signe (-) la valeur indiquée dans le champ Dimension H (hauteur).

Remarque : Dans une disposition classique comme celle-ci, Patternshop permet de déplacer les motifs en utilisant les flèches du clavier, et ainsi contrôler très précisément les écarts entre chaque motif de manière à trouver la disposition idéale.

Vous pouvez ensuite passer à l'étape 4 pour manipuler la matrice.
