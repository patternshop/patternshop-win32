Vocabulaire
Vous trouverez ici une définition des termes essentiels utilisés dans Patternshop.

  • Motif (a) : Image répétée qui constitue le réseau.
  • Matrice (b) : Cadre modifiable dans lequel sont placés les motifs, et qui définit leur disposition dans le réseau.
  • Image libre : Image non répétée, positionnable n'importe où sur le document.
  • Réseau (c) : Espace illimité, formé par la répétition du ou des motifs contenus dans les matrices.
  • Document (d) : Zone définie d'un réseau.
  • Patron : Forme prédéfinie sur laquelle est appliqué le réseau.
